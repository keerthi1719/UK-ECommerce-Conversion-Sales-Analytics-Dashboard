# 📊 UK E-Commerce Conversion & Sales Analytics Dashboard

## Overview

This Power BI project provides a comprehensive analysis of UK e-commerce operations by tracking sales performance, customer behavior, product profitability, fulfillment efficiency, and conversion metrics. The dashboard helps businesses gain actionable insights and make data-driven decisions.

## 🚀 Key Features

- Revenue and Profit Analysis
- Customer Conversion Funnel
- Customer Segmentation (RFM Analysis)
- Product Performance Tracking
- Return Rate Analysis
- Sales & Fulfillment Monitoring
- Regional Performance Insights
- Interactive KPI Dashboard

## 📈 KPIs Tracked

- Total Revenue: £1.98M
- Gross Profit: £989K
- Total Orders: 10K
- Total Customers: 500
- Conversion Rate Analysis
- Average Delivery Days
- Product Return Rate


## 🔍 Key Insights

- London generated the highest revenue among all regions.
- Diamond loyalty customers delivered the highest customer value.
- High-value customers contributed significantly to overall revenue.
- Premium products outperformed other categories.
- Return rates increased during seasonal sales periods.

## 🛠️ Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization



## 🎯 Project Outcome

This dashboard enables stakeholders to monitor business performance, optimize customer retention, improve product strategies, and support data-driven decision-making through interactive analytics.
